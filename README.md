# An example `rocqblueprint` project

An example showcasing the utilization of the
[`rocqblueprint`](https://www.github.com/reiniscirpons/rocqblueprint) tool
to a simple Rocq project.

The automatically generated homepage for this project is available at
[https://reinisc.id.lv/example-rocq-blueprint-project/](https://reinisc.id.lv/example-rocq-blueprint-project/),
see especially the [web blueprint](https://reinisc.id.lv/example-rocq-blueprint-project/blueprint/)
and [interactive dependency graph](https://reinisc.id.lv/example-rocq-blueprint-project/blueprint/dep_graph_document.html)


See the [`blueprint/src/content.tex`](./blueprint/src/content.tex) for the underlying latex source
used to generate the blueprint. The comments in this file give a brief explanation of the
`rocqblueprint` macros and their use.

See the [`.github/workflows/blueprint.yml`](./.github/workflows/blueprint.yml)
file for an example github workflow configuration.
